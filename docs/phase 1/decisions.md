# ADR-0001 Tech-Stack

**Entscheidung:** Next.js 14 (App Router), Supabase (DB+Auth), Twilio (Voice), OpenAI Whisper/GPT, ElevenLabs (TTS), Google Calendar, Stripe.

**Begründung:** schnelle MVP-Entwicklung, geringe Betriebskosten, EU-Hosting möglich, gute DX.

**Alternativen geprüft:** Firebase, Vonage, PlayHT – verworfen wegen DX/Preis/Deutsch.
